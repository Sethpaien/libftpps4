## FTPPS4
FTP Server for the PS4.

## Compiling
Prerequisites:
* [ps4sdk] (https://github.com/CTurt/PS4-SDK)
Alter Makefile to change TEXT and DATA address

Once all the prerequisites have been met, you can compile this homebrew by typing:
```
	$ make
```
